Ghibli API

Has 5 endpoints. Each endpoint will give you all of the options, and you can get more specific ids from there. 
    Films  
    People
    Locations
    Species
    Vehicles 

So, you can get a list of all the films, with specific ids. 
Each category goes, category/<id>
